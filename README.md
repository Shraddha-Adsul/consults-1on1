# Doctor Consults Landing Page

A simple vanilla HTML/CSS/JavaScript landing page for one-on-one doctor consultations.

## Features

- Responsive grid layout of available doctors
- Doctor profile cards with key information
- "Book Consult" functionality for each doctor
- "View Profile" modal with detailed doctor information
- Mobile-friendly design
- No frameworks or dependencies required

## Files

- `index.html` - Main landing page
- `styles.css` - Styling
- `script.js` - Interactive functionality

## Usage

Simply open `index.html` in a web browser to view the landing page.

## Customization

Edit the doctor data in `script.js` to add/modify doctor profiles. Update the HTML in `index.html` to add more doctor cards to the grid.
